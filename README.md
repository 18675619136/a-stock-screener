# A股全市场筛选脚本 📊

基于新浪+腾讯公开API的A股筛选工具，筛选小市值、小流通股本、且符合技术面买入条件的个股。

## 筛选条件

1. **排除ST/退市股**
2. **流通股本 < 10亿股 且 总市值 < 1000亿**
3. **MA5 > MA20（金叉）且 收盘价 ≤ MA20 × 1.10**

## 使用方式

```bash
# 全量运行
python3 screen_v2.py

# 快速测试（前30只）
python3 screen_v2.py --limit 30
```
